### Features of Asset Management Web App:

- Join as HR/Admin
- Join as Employee
- Payment System Integration
- View Asset List
- Add Assets
- View All Requests
- View Custom Requests List
- View Employee List
- View Profile
- View Team List
- Request for Asset
- Make Custom Request
- User Authentication using Firebase and JWT
- JWT Authentication on Private Routes and Admin Routes

### Technologies which have been used in this app:

- React
- React-Router
- Firebase
- Node.js
- Express.js
- MongoDB
- Stripe.js
